# Marlu Weekly Timesheet Automation Form

A Flask-based employee Exit / Return form system designed for workforce attendance tracking and SharePoint integration.

## Features

- Employee Sign Out / Sign In form
- Supervisor selection
- Site and position display
- SharePoint list integration
- Validation for duplicate sign-outs
- Missing return detection
- OPMS employee mapping
- Real-time attendance record submission
- Gap tracking support for timesheet reconciliation

## Technologies Used

- Python
- Flask
- Microsoft Graph API
- SharePoint Online
- Azure App Service
- HTML / CSS
- GitHub
